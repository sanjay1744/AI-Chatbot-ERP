Gmail Integration Guide

If you are integrating with a Google/Gmail account, standard passwords will be rejected. You must configure a secure Google App Password:

  1. Go to your Google Account security page.
  2. Enable 2-Step Verification if you haven't already.
  3. Search for App Passwords in the search bar.
  4. Select "Other" or "Mail" and generate a 16-character password code.
  5. Use this generated 16-character code as the IMAP & SMTP password fields.

Standard GMail Settings:

  • IMAP Server: imap.gmail.com
  • IMAP Port: 993 (SSL)
  • SMTP Server: smtp.gmail.com
  • SMTP Port: 465 (SSL)
